Fast free publish options:
1. GitHub Pages
   - Create a GitHub repo
   - Upload all files in this folder
   - In repo Settings > Pages, enable Pages from the main branch root
   - Open the live URL on your iPhone in Safari
   - Tap Share > Add to Home Screen

2. Netlify Drop
   - Zip this folder
   - Drag the zip/folder into Netlify's manual deploy page
   - Open the live URL on your iPhone in Safari
   - Tap Share > Add to Home Screen

Notes:
- iPhone Safari does not support true silent file overwrite from a local HTML file the way desktop Chrome can.
- Once hosted and opened in Safari, the app's browser storage works much more reliably on iPhone.
- Use Export JSON for backups.
